1.2. CSS3(Cascading Style Sheets) tiykarları.

CSS birinshi ret 1994 jıl 10 -oktyabrde Xakon vium Li tárepinen usınıs etilgen. CSS Jumısshı toparı CSS 1-dárejesinde shehilmegen mashqalalardı shehiwdi basladı, nátiyjede CSS 2-dárejesi 1997 jıl 4-noyabrde jaratıldı. 1998 jıl 12-may kúni ol W3 C usınısı retinde baspa etildi. 1998 jılda baslanǵan CSS 3-dárejesi 2014 jıldan baslap ele da islep shigılıp atır.

CSS (ing. Cascading style sheets, uzb. Kaskadlangan stiller kestesi) - bul belgilew tilinen paydalanıp jazılǵan hújjetlerdiń kórinisin súwretlew ushın isleti-letiletuǵı rásmiy til esaplanadı. Ol tiykarınan HTML hám XHTML belgilew tilleri járdeminde jazılǵan web - betlerdiń kórinisin súwretlew hám proektlestiriw quralı retinde isletiledi, biraq hár qanday XML hújjetlerine, mısalı, SvG yamasa XUL-ge de qollanılıwı múmkin.

CSS web - bet jaratıwshıları tárepinen reńler, shriftler, bólek bloklardıń rejimin hám bul web - betlerdiń payda bolıwınıń basqa ji-hatlarini belgilew ushın isletiledi. CSS ni islep shıǵıwdıń tiykarǵı maqseti web - bettiń (HTML yamasa basqa belgilew tilleri járdeminde islep shıǵarılǵan ) logikalıq dúzilisiniń xarakteristikaın bul web - bet (házir rásmiy CSS tili járdemi-de islep shıǵarılǵan ) sırtqı kórinisi xarakteristikaınan ajıratıw edi. Bul ajıratıw hújjettıń bar ekenligin asırıwı, kóbirek maslasıwshılıǵı hám prezentaciyanı basqarıw qábiletin támiyinlep, sonıń menen birge strukturalıq quramdaǵı quramalılıq hám tákirarlanıwshańlıqtı kemeytiwi múmkin.
