1.3. CSS Sintaksisi.

CSS ápiwayı sintaksiske iye hám túrli stilistik ayrıqshalıqlardıń atların anıqlaw ushın bir neshe anglichan gilt sózlerden paydalanadı. Usıllar kestesi qaǵıydalar diziminen ibarat. Hár bir qaǵıyda yamasa qaǵıydalar kompleksi bir yamasa bir neshe selektordan hám deklaratsiya blokgınan ibarat.

 CSS qaǵıydaları rásmiy CSS tilinde jazılǵan. CSS formatı bul CSS qaǵıydaları hám olar haqqındaǵı túsindiriwlerdi óz ishine alǵan tekstli fayl esaplanadı.

 CSSda da járiyalaw bólimi ámeldegi bolıp, ádette onı jazıw shárt emes. Bul bólimde nizamlıqlar qanday jazılǵanlıǵı hám qaysı tárzde nátiyjeni ámelde qollanıw ámel etiw kerekligi kórsetiledi.

 Sintaksisi tómendegishe:

@buyrıq nızamlılıqları

Mısal:

@charset “utf-8”

Túsindirme retinde jazılǵan yamasa brauzer itibar qaratpaslıq kerek bolǵan jazıwlar arnawlı belgiler «/*» hám «*/» lardıń arasına alıp qóyıladı.

 Ulıwma alǵanda CSS tómendegi sintaksis tiykarında aytıladı:

Atamalar {qaǵıydalar}

Bunday atamalar:

Atamalar [, atamalar]

Qaǵıydalar:

Qaǵıyda [, qaǵıyda]

Qaǵıyda :

Qásiyet: mánisler

Mánisler:

mánis [, mánis ]

 

Daǵaza etiw bólimindegi «nızamlıqlar» kórinisler «qaǵıydalar» kórinisinde hám beriliwi múmkin. Máselen arnawlı qásiyet qosıw:

@font-face {

font-family: Consolas; /* Xusnixat ati */

src: url(/font.ttf); /*Xusnixat faylınıń mánzili */

}

 

Atamalaw negizinde kelisilgen identifikator bolıp, onı ekinshi belgilewden baslap sızıqsha (-) qatnasıwǵa ruxsat etiledi. Bul atamalawlardı toparlawda paydalanıladı. Atamalaw tómendegi maǵlıwmatlardan birin ańglatıwı múmkin:

o   Universal atamalaw.

o   Tegning atı.

o   Tegning «id»si.

o   Klasslaw.

o   Ishindegi tegler.

o   Qońsı teg.

o   Aǵayin teglar.

o   Ishki tegler.

CSS stilleridi tórt usıl menen web-hujjetke kiritiliw yaki jaylastırılıwı múmkin.

1.     Stiller xarakteristikası bólek faylda bolǵanda, ol <head> elementine kiritilgen <link> elementinen paydalan hújjetke jalǵanıwı múmkin:

<!DOCTYPE html>
<html>
   <head>
      .....
     <link rel="stylesheet" type="text/css" href="style.css">
   </head>
   <body>
      .....
   </body>
</html>

 

2.     Eger stil kesteleri tiykarǵı hújjetten bólek jaylastırılsa, ol hújjetke <style> elementindegi @import kórsetpesi menen jalǵanıwı múmkin:

<!DOCTYPE html>

<html>

   <head>

      .....

      <style media="all">

         @import url(style.css);

      </style>

   </head>

</html>

3. Hújjet ishinde stiller xarakteristikalanǵanda, olar <head> elementine kiritilgen <style> elementine kiritiliwi múmkin.

<!DOCTYPE html>

<html>

   <head>

      .....

      <style>

         body { 

            color: red;

         }

      </style>

   </head>

   <body>

      .....

   </body>

</html>

3.     Hújjet ishindegi elementtiń attributı sıpatında kiritiliwi múmkin:

<!DOCTYPE>

<html>

   <head>

      .....

   </head>

   <body>

      <p style="font-size: 20px; color: green; font-family: arial, helvetica, sans-serif">

         .....

      </p>

   </body>

</html>

 

"CSS" qaǵıydalar toplamınıń tiykarǵı sintaksisi eki bólimnen ibarat: selektor hám bir topar qaǵıydalar, olardıń hár biri qásiyet atı hám sol qasiyettiń mánisinen ibarat.

selector {  

   property: value;

}

 

Aldın selektor jazıladı hám keyin qaǵıydalar {qawıslar} ishine jazıladı. Hár bir qaǵıydanıń ózgesheligi hám mánisi bir birinen: eki noqat penen ajratıladı hám qaǵıyda mudamı ; noqatlı útir menen tawsıladı.
