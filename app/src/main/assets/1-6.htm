1.6. Miyras alıw.

Miyras - CSStiń tiykarǵı ózgesheliǵi – bul tiykarǵı nusqa hám keying nusqalar qatnasıqlarına tayanadı. Miyrasxor - bul ózgeshelikler tek ǵana kórsetilgen elementke emes, bálkim onıń áwladlarına da nátiyjeni ámelde qollanıw mexanizmi.  Miyras HTML iyerarxiyası bolǵan hújjet daraxtına tiykarlanǵan. Uyalar jaylastırıwǵa tiykarlanǵan betlerdegi elementler. Tiykarǵı elementler CSS ózgesheliklerin óz ishine alǵan hár qanday tiykarǵı elementten miyras etip alıwları múmkin. Ádette, balalar elementleri tekst penen baylanıslı ózgesheliklerdi miyras qılıp aladı, biraq olardıń blok penen baylanıslı ózgeshelikleri miyras qılıp alınbaydı.

Miyras qılıp alınıwı múmkin bolǵan ózgeshelikler: reń, shrift, háripler aralıǵı, qatar biyikligi, dizim metodı, teksttiń izleniwi, teksttiń kiritiliwi, teksttiń ózgeriwi, kórinisi, bos jayları hám sózler aralıǵı.

Miyras qılıp alınbaytuǵın ózgeshelikler: fon, shegara, displey, júziwshi hám tınıq, bálendlik va keńlik, shegara, min hám maksimal bálentlik hám keńlik, kontur, tashıp ketiw, toltırıw, jaylasıw, tekstti bezetiw, vertikal tegislew hám z-indeks. Ózińizdiń tablicańızda bazı bir ózgesheliklerdi qayta-qayta daǵaza qılıwdan qashıw ushın miyrastan paydalanıwıńız múmkin, bul bolsa qısqartırılǵan CSSke imkaniyat beredi.

CSSde miyrasxorlıq klassqa tiykarlanǵan dástúrlew tillerindegi miyras penen birdey emes, bul jerde siz B klasstı "A klassqa uqsap, lekin ózgerisler menen" belgileniwińiz múmkin. CSS járdeminde elementti "A klassı menen ózgertiriw múmkin, biraq ózgerisler menen". Biraq, keyinshelik ózgerislerdi tákirarlamastan bir neshshe elementlerdi isletiw ushın paydalanıwı múmkin bolǵan B klasstı CSS ti anıqlaw múmkin emes.

Mısal:

h1 {

   aqshıl qızǵısh reń;

}

<h1> Bu <em> miyrastı súwretlew ushın <em> em> </ h1>

 

Eger em elementine reń berilmese, ajratılǵan sóz "illustrate" onıń tiykarǵı h1 elementi reńin miyras etip aladı. H1 usılı aqshıl qızǵısh reńge iye, sonıń ushın em elementi de aqshıl qızǵısh reńge iye.

Kóp hallarda bir obyekt ushın tayınlanǵan qaǵıydalardan úlgi alıp, basqasha atalıwı shártler kompleksin jaratıwǵa tuwrı keledi. Bunda atalıwdıń ózin shaqırıp qoyıw jetkilikli:

.title, h1,h2 {

border: 1px solid #ccc;

border-raduis: 3px;

padding: 2px 10px;

color: #333;

}

h1{

font-size: 18px;

}

h2 {

font-size: 16px;

color: maroon;

}
