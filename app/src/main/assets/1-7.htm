1.7. Bootstrap framework.

Bootstrap - web-sayttı tez hám ańsat islew ushın eń belgili va eń kushli (HTML, CSS hám JavaScript) maslasıwshı framework esaplanadı.

Bootstrap - web sayttı islep shıǵıw tez hám ańsat bolǵan kushli front-end (old) framework bolıǵ tabıladı. Bul HTML hám CSS-ke tiykarlanǵan forms, buttons, navigations, dropdowns, alerts, modals, tabs, accordions, carousels, tooltips hám basqalar sıyaqlı ulıwma paydalanıwshı interfeysi strukturalıq bólimlerdi jaratıw ushın mólsherlengen dizayn shablonların óz ishine aladı. Birinshi Web ñ betimizdi Bootstrap járdeminde jaratamız. Biz Bootstrap CSS hám JS fayllardı sondayaq, Javascript: JQuery hám Popper.js dı CDN sıyaqlı basqa qarsılıqlardı qosıp bir tiykarǵı Bootstrap shablon jaratamız.

Biz Bootstraptı CDN (Content Delivery arqalı joybarıńıǵa qosıwdı usınıs etken bolar edik. Sebebi CDN júkleniw waqtın azaytadı, sebebi olar fayllardı dúnya boylap kóplegen serverlerge jaylasadı qashanda paydalanıwshı faylǵa soraw jibergende, oǵan ózlerine jaqın (uqsas) bolǵan server xızmetinen paydalanıǵa urınadı. Úlgileremizde de hátte CDN siltemelerdi isletpekshimiz: Keliń, tómendegi qádemlerdi kórip shıǵamız.

1-qádem: Tiykarǵı HTML fayl jaratıw

Bos ayna menen baslaymız hám tómendegi kodtı jazıp basic. html retinde saqlay-miz.

<!DOCTYPE html>

<html lang="en">

<head>

††† <meta charset="utf-8">

††† <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

††† <title>Asosiy HTML fayl</title>

</head>

<body>

††† <h1>Hello, world!</h1>

</body>

</html>

 

Mudamı viewport <meta> tegini <head> bólegi ishine mobil apparatlarda web-sayttı úlkeytiw hám mobil apparatlarǵa maslastırıwdı támiyinlew ushın jazıladı.

2-qádem: HTML fayldı Bootstrap shablonına aylandırıw

†Joqarıda isletgen HTML fayldı Bootstrap shablonına aylandırıw ushın, ápiwauı Bootstrap CSS hám JS fayllardı sonıń menen birge, talap etilgen JQuery hám Popper.js tı olardıń CDN siltemeleri arqalı qosıp alamız. Javascript faylların betmizning tómengi bólegine qosıwımız kerek, anıqrog'i </body> tegin jabılıwınannan aldın qosıw web - betmizni islew iskerligin jaqsılaydı. Tómendegi sıyaqlı :

Barlıǵın ornatdıq! Bootstraptıń CSS hám JS faylların hám talap etilgen JQuery hám Popper.js tı qosıp bolǵanımızdan keyin, hár qanday sayt yamasa qosımshanı Bootstrap frameworki járdeminde jasawımız múmkin. integrity hám crossorigin atributları CDN siltemelerge Subresource Integrity (SRI, ózbekshe Resurslar pútinligi) ni ámelge asırıw uchın qosılǵan. Bul saytimizga bolatuǵın hújim qáwpin kemeytiw ushın isletiledi.

3-qádem: Saqlaw hám nátiyjeni kóriw

Keleside biz jumısshı stolımızǵa " bootstrap-template. html" fayldı saqlaymız. Web brauzerde fayldı ashıw ushın, oǵan baǵdarlaymız, keyin ústine oń tuymeni basamız hám brauzerin tańlaymiz. Alternativ túrde, biz brauzerdi ashıp fayldı brauzerge qoyıw arqalı da kóremiz.

Bootstrap paydalanıwdıń tiykarǵı abzallıǵı ñ dinamikalıq til usıllar sezilerli CSS múmkinshiliklerin keńeytiredi. Sonıń menen birge, ózgeriwshiler, ishki ústinler jaratılıwımız múmkin.

Bootstrap tiykarǵı quralları:

Tor - mısalı, 90 px ústin keńligin biz CSS qásiyetleri hújjette. span2 klasıına, degen mánisnida paydalanıwımız múmkin;

Shablonlar - turaqlı yamasa rezina ulgi hújjet;

Tipografiya - shrift xarakteristikası, tırnaq, shriftler ushın birdey klasslar tariypi hám taǵı basqa ;

Media - suwretler hám videolardı baqlawdıń bar ekenligi;

Navigatsiya - tabs, betler, menyular hám avtomobil kompleksi ushın rásmiylestiruvi sabaqları ;

Alerts - dizimge alıw sáwbetler, talap hám pop-up áynekler.
