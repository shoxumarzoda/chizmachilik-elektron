2.1. JavaScript texnologiyası haqqında qısqasha maǵlıwmat

JavaScript 1995-jıl 4-dekabrde payda bolǵan, sol kúni til óz atınıń alǵan. Bunnan aldın ol LiveScript dep atalǵan hám Netscape Communications hám Sun Microsys-tems tárepinen birgelikte elementlerdi basqarıw hám web-resurslarınıń strukturalıq bólimleri - súwretler, plaginlar, Java programmaları hám web-betlerdi jaratıwda isleti-ladigan basqa elementlerdiń óz-ara tásirin támiyinlew ushın jaratılǵan.

JavaScript ta C tiline uqsas sintaksis ámeldegi, biraq ol jaǵdayda bir qatar úlken áhmiyetke iye ayırmashılıqlar ámeldegi:

-       Ob'yektlar menen islew múmkinshiligi, sonday-aq programmanı orınlaw waqtında obiekttiń túrin hám dúzilisin anıqlaw.

-       Funksiyalardı parametrler retinde uzatıw hám qaytarıw (return), sonıń menen birge olardı ózgeriwshige belgilew múmkinshiligi.

-       Avtomatikalıq turni salıstırıw mexanizminiń bar ekenligi.

-       Anonim funksiyalardan paydalanıw.

JavaScript Járdeminde ádetde maǵlıwmatlı hám baylanıs áyneklerin shıǵarıw, animatsiyalardı kórsetiw sıyaqlı wazıypalardı orınlaw múmkin. Bunnan tısqarı, JavaScript-ssenariy geyde ózi islep turǵan brauzer hám platforma tipini anıqlaw múmkin.

 JavaScript-ssenariylar paydalanıwshı tárepinen kiritilip atırǵan maǵlıwmatlardı tuwrılıǵın tekseriwde de qolay esaplanadı.

 JavaScript tili házirgi kúnde eń ataqlı tillerden biri esaplanadı. Kóp reytinglerde ataqlılıǵı boyınsha birinshi orında turadı. Bul til arqalı oyın jaratılıwmasımız, web-programmalar jazıwımız, mobil qosımshalar, kompyuter ushın programmalar, hullas hár qanday túrdegi programmalar jazıwımız múmkin. Brauzerlerde bolsa JavaS-cript tiykarǵı programmalastırıw tili esaplanadı. Lekin, eń ataqlı til degeni - eń qolay til degeni emes. JavaScript de qandayda bir bir qıyınlaw programma jaratpaqshı bolsańız - albette kitapxana, yamasa frameworklardan paydalanasız. Kitapxana hám freymwork bul aldınan jazıp qoyılǵan kodlar kompleksi.

 Hár bir HTML hújjet ob'yekt bolıwı múmkin hám ol tómendegishe xarakter-lenedi:

-       qásiyetlerı;

-       metodlar (usıllar -programmalar -funktsiyalar );

-       hádiyselar.

Ob'yektiv modeldi Web - bet hám brouzerlerdiń baylanıs usılları kórinisinde sáwlelendiriw múmkin. Ob'yektiv model - bul ob'yektlar, usıllar, ayrıqshalıqlar hám basqa elementlerdiń brouzer programmalıq támiynatında kórinetuǵın bolıwı bolıp tabıladı. Ob'yekt - bul JavaScriptdasturiy tiliniń berilgen elementleri ma`nisiniń turi bolıp tabıladı. Birdey túrdegi birdey ózgeshelikke iye bolǵan, birdey usılǵa iye bolǵan, birdey xodisaga iye bolǵan ob'yektlar bir klasqa birlesedi (klass). Klasslar - bul ob'yektlarning múmkin bolǵan redaktorı bolıp tabıladı.

Javascript <head> yaki <body> ishinde

Bir qansha scriptlerdi HTML hújjetke jaylawımız múmkin. Scriptler, HTML bettiń <body> yamasa <head> bólimleriniń birinde yamasa ikkovida jaylasıwı múmkin.

Javascript <head> ishinde

Tómendegi úlgide, Javascript funksiya HTML bettiń <head> bóleginde jaylasqan.

 Tuyme basılǵanda funksiya shaqırıladı :

<!DOCTYPE html>
<html>

<head>
<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraf ózgertiriledi.";
}
</script>
</head>
<body>

<h1>Web beti</h1>
<p id="demo">Paragraf</p>
<button type="button" onclick="myFunction()">Try it</button>

</body>
</html>

 

Javascript <body> ishinde

Tómendegi úlgide, Javascript funksiya HTML bettiń <body> bóleginde jaylasqan.

 Tuyme basılǵanda funksiya shaqırıladı:

<!DOCTYPE html>
<html>
<body>
<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>
<script>
function myFunction() {
 document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</body>
</html>

 

Scriptler <body> tegining aqırına jaylastırılsa, bet ashılıw tezligi asadı, sebebi Script interpretatsiyasi betti ashıw tezligin pasaytiradi. Yaǵnıy, body bólekke jazsak, aldın headdagi stiller shaqırıladı, bunda web - bet tolıq kórinisi ashılǵannan keyin, javascript fayllar juklenedi.

Eksternal (sırtqı ) Javascript

 Scriptler eksternal (sırtqı ) fayllarda da jaylasıwı múmkin:

Eksternal fayl: myScript.js

function myFunction() {
 document.getElementById("demo").innerHTML = "Paragraf ózgertiriledi.";
}

 

Eksternal Scriptler kóplegen hár qıylı web - betlerde birdey kod isletilingende ámelde qollanıladı. Javascript fayllarında.js keńeytpe atı bar.

Eksternal Scriptti isletiw ushın, <script> tegining src (source: derek) atributına Script fayl atınıń jaylaw jetkilikli:

<script src="myScript.js"></script>

 

Eksternal Script siltemelerdi <head> yamasa <body> ishine jaylawımız múmkin. Eksternal Script fayllarında <script> tegleri isletilmeydi. Tek ǵana javascript kodın jazıw jetkilikli.

Eksternal siltemeler

Eksternal Scriptlerge tolıq URL mánzil arqalı yamasa ámeldegi web - betke salıstırmalı mánzil arqalı silteme etiliwi múmkin.

Scriptke tolıq URL mánzildi isletiw:

<script src="https://www.tutorials.uz/js/myScript1.js"></script>

 

Bul úlginde bolsa, ámeldegi web - bette ámeldegi bolǵan papkada jaylasqan Script adresi qollanıladı:

<script src="/js/myScript1.js"></script>
