2.2. JavaScript programmalastırıw tili operatorları.

JavaScript tiliniń tómendegi operatorların kórip shıǵamız : var; {... }; if;

while; for; for... ın; break; continue; return. Bul dizim tolıq emes.

Operator var

Var operatorı ózgeriwshin ańlatıw ushın hizmat etedi. Sonıń menen birge, ózgeriwshi hár qanday orınlanıp bolınǵan maǵlıwmatlar túrleri mánisin qabıllawı múmkin. Ámelde kóbinese var qollanılmaydı. Ózgeriwshi túrge muwapıq ózlestiriwdiń ańsat jolı menen payda etiledi:

var a

var a=10

var a = new Array ()

var a = new Image ()

 

Joqarıda keltirilgen barlıq mısallarda var tuwrı paydalanılǵan hám JavaScript programmasında qollanılıwı múmkin. Ózgeriwshi paydalanılıp atırǵan háreket salasında ózgeriwshi blok arqalı anıqlanadı. Ózgeriwshiniń maksimal háreket tarawı - bet bolıp tabıladı.

Figurali qawıslar {...}

JavaScript bir neshe ańlatpanı bir ańlatpaǵa birlestiriw imkaniyatın beredi.

Bunday ańlatpa bloklı ańlatpa yamasa blok dep ataladı. Onıń ushın kerekli kod figuralı qawıs {} ishine alınadı.

a =11

{b = “12”

c=a-b

}

Bloklardan quramalı ańlatpalar, atap aytqanda, shártli operatorlarǵa iye ańlatpalar dúziw ushın paydalanıladı.

Shártli operator if

Shártli operator programmanı tarmaqlanıwı ushın bazı bir logikalıq shártlerde

isletiledi. Ulıwma sintaksis:

if (mantiqiy ifoda) operator1

[else operator2

]

 

Logikalıq ańlatpa — bul sonday ańlatpaki, ol true yamasa false mánisin qabıl qıladı. Eger ol teń bolsa, true 1 operator ámelge asırıladı. Kvadratlıq qawıslarda strukturalıq operator if tiykarǵı tarmaqtı esaplaw alternativi shárt emes:

if (navigator. appName =="Netscape")

{window.location.href="http://ziyo.edu.uz/netscape.html";

}

else {

window.location.href="http://ziyo.edu.uz/explorer.html";

}

 

Tarmaqlaw operatorı arnawlı bir hádiyse júz bergeninde koddıń arnawlı bir bólegin orınlaw imkaniyatın beredi. Ózgeriwshige arnawlı bir ańlatpanıń beriliwi yamasa arnawlı bir ańlatpanı esaplawdıń ol yamasa bul tárizde bólek munasábet bildiriliwi lazım bolǵan nátiyjesi mine sonday hádiyse bolıp, xizmet etiwi múmkin. Tarmaqlaw operatorı tómendegi formatqa iye:

if ({Shart}) ... ―ol jaǵdayda‖ blogı

[else

... ―keri jaǵdayda ‖ blogı]

 

Shárt – Logikalıq ańlatpa bolıp, oǵan muwapıq interpretator qaysı bloktı orınlaw tuwrısında qarar qabıl etedi. Eger shárt true («haqiqat») ańlatpasına iye bolsa, “ol halda” blokı atqarıladı. Eger shárt false («yolg'on») ańlatpasına iye bolsa, “keri jaǵdayda” blokı orınlanadı (eger ol ámeldegi bolsa ). Bardı da «keri japdayda» blogı ámeldegi bolmasa, programmanıń náwbettegi ańlatpası atqarıladı. Eger shárttiń nátiyjesi null yamasa undefined bolsa, tarmaqlaw operatorı false ge qanday munasábet bildirgen bolsa, oǵan da sonday munasábet ańlatadı. Mısal:

if (x == 1) {f = 3

h = 4

} else {f = 33

h = 44

}

Operator while

While udayı tákirarlanatuǵın operator do-while udayı tákirarlanatuǵın operatorın yadǵa saladı, biraq shárt udayı tákirarlanatuǵın operator denesi atqarılıwınan aldın tekseriledi. Sol sebepli, eger ol aldınboshdan jalǵan bolsa, udayı tákirarlanatuǵın operator qandayda bir marte da atqarılmaydı.

Operator while dáwirdi anıqlaydı. Ulıwma halda tómendegishe anıqlanadı:

While (logikalıq ańlatpa)

operator;

Operator hám struktura (sostav) - dáwir denesi esaplanadı. Dáwir denesi logik

shártler tuwrı bolǵandaǵana ámelge asıriladı:

while (flag==0)

{id=setTimeout ("test ();",500);}

 

Tiykarınan bul tipdagi dáwir bazı bir hádiyselerdiń udayı tákirarlanatuǵın (periodicheskiy) háreketlerdi orınlawdan aldın qollanıladı.

 Mısal ushın:

while (a < 100) {a=a*i+2

i=++i

}

 

Operator for ... in

{

document.write (v+"<BR>");

}

 

 

break hám continue operatorları

Geyde cikldıń atqarılıwın toqtatıw talap etiledi. Onıń ushın JavaScript programmistlerge break hám continue operatorların usınıs etedi.

break operatorı siklning atqarılıwın toqtatıw hám hám odan keyingi ańlatpaǵa

 ótiw imkaniyatın beredi.

while (a < 100) {a=a*i+2

if (a > 50) break

i=++i}

 

Bul mısalda biz, eger a ózgeriwshiniń ma`nisi 50 den aspasa, udayı tákirarlanatuǵın  operatordıń atqarılıwın (biraq programmanıń atqarılıwın emes) toqtatamiz.

Сontinue operatorı udayı tákirarlanatuǵın operatordı qayta júklew, yaǵnıy udayı tákirarlanatuǵın operatordı

 denesine kiretuǵın barlıq keyingi komandalardı atqarmastán qaldırıw jáne onıń basına qaytıw imkaniyatın beredi. Shárt tekserip kóriledi (for ciklı, bunnan tısqarı, inkrement ańlatpasın atqaradı ) hám, eger shárt haqıyqıy bolsa, udayı tákirarlanatuǵın operator denesi basınan atqarıladı.

while (a < 100) {i=++i

if (i > a && i < 11) continue

a=a*i+2

}

Bul jerde biz i dıń 10 nan 20 ǵa shekem bolǵan barlıq ańlatpaları ushın a nıń

esaplanǵan barlıq ańlatpaların ótkeremiz.

break hám continue operatorlarınıń tolıq formatı tómendegi kóriniske iye:

break / continue [{Belgi}]

Bul jerde belgi áyne break yamasa continue operatorı qollanilayotgan cikldı

belgileydi. Belgi tómendegi formatqa iye:

{Belgi atı} : ... Ańlatpa

 

Dástúrge túsindirme

JavaScript iqıiyarımizǵa túsindirme kirgiziwdiń eki operatorın beredi.

//... Túsindiriw qatarı

Bul operator ańlatpa aqırına bir qatarlı túsindirmeni kirgiziw imkaniyatın beredi.

a=b+c; // Bul – bir qatarlı túsindirme

 

Itibar bergen bolsańız, túsindirme ańlatpa tawsılǵanın kórsetiwshi noqat hám útir (;) den keyin jaylasqan.

/*

…túsindirme

*/

Bul operator bolsa programma kodına hár qanday shamadaǵı túsindirmeni kirgiziw imkaniyatın beredi.

/* Bul ańlatpada biz eki ózgeriwshiniń ańlatpasın qosamız hám nátiyjeni

 úshinshi ózgeriwshige jaylastıramız.

*/

a=b+c;
