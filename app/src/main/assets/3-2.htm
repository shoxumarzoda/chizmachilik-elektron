3.2. PHP dástúrlew tilindegi tiykarǵı sintaksisler.

Instruktsiyanı  bir nеshshe bólimge bólip  kórip shıǵamız,  yaǵnıy:

Ø túsindirme (kommеntariy)ler jaratıw;  

Ø ózgeriwshiler;  

Ø ózgermes (konstanta)lar;  

Ø maǵlıwmat  túrleri;

Ø opеratorlar.

Túsindirme (kommеntariy) ler jaratıw

PHP de túsindirme jazıwdıń eki usılı bar.

1. Eki slesh (//) arqalı onnan keyingi qatar tolıq túsindirme boladı jáne bul tek sol qatarǵaǵana tásir etedi:

// Bul túsindirme qatarı

Bul túsindirme jazıwdıń ájayıp túri bolıp, aljasıqlar berip atırǵan qatardı waqtınsha saqlap turıw imkaniyatın beredi:

// echo "X stringi $x qa teń";

Sonıń menen birge, túsindirmeniń bul túrin qandayda bir kod qatarı qanday islewin túsindiriw ushın da kirgiziwińiz múmkin:

$x += 10; // $x 10ǵa inkrement

Eger sizge bir neshhe qatarlı túsindirme kerek bolsa tómendegi kodtan paydalanasız:

<?php

/* Bul bir neshhe qatarlı túsindirme

jazıw ushın */

?>

 

Pútkil baslı kodlı fayldı waqtınsha /* */ arasına jaylastırıp qoyıw da múmkin.

 

Ózgeriwshiler

Ózgeriwshiler $ belgisi menen baslanadı. «Sálem, men – PHP Scriptpen!» sózin tómendegi jol menen de payda etse de boladı:

$jazıw = “Sálem, men – PHP Scriptpen!”;
echo $jazıw;
?>

Nátiyje:

Sálem, men – PHP Scriptpen!

 

Bunnan tısqarı, ózgeriwshilerge shaqırıq etiwińiz hám olardıń mánisin ózgertiwińiz múmkin.

$var1 = 14;

$var1 = $var1 + 1;

print $var1;

Nátiyje:

15

 

Ózgermes (konstanta)lar

Ózgermesler ózgeriwshilerge uqsap ketedi, biraq ol belgilengennen keyin ózgertirip bolmaydı. Konstanta - ápiwayı baha ushın identifikator esaplanadı. Bahanı Script (kodlaw processinde) dawamında ózgertirip bolmaydı. Ózgeriwshilerden ayrıqsha bolıp esaplanıw, Konstantalar avtomatikalıq túrde pútkil Scriptte global ortalıqta boladı.

Konstanta jaratıp alıw ushın biz define () funksiyasın qollaymiz.

Define (name, value, case-insensitive)

Parametrleri:

  • name(atı): Konstantanıń atın belgileydi
  • value(mánis): Konstantanıń mánisin belgileydi
  • case-insensitive (Úlken-kishi hárip sezbeydi): Case-insensitive (Úlken-kishi háriplerdi sezbeytuǵın) at bolıwı kerekligin belgileydi. Standart FALSE ǵa teń.

PHPde maǵlıwmatlar túri

PHP 8 de maǵlıwmatlar túrlerin qollap quwatlaydı:

Tórt skalyar túr:

– boolean (ekilemshi maǵlıwmat)
– integer (pútin san)
– float (ózgerip turıwshı noqtalı san yaki ‘double’)
– string (qatarlar)

Eki aralas túr:

– array (massivler)
– object (obyektler)

Eki arnawlı tur:

resource (resurslar)
NULL («bos»)

Bir neshshe hár túrlileri de bar:

– mixed (aralas)
– number (san)
– callback (qayta shaqırıw)

Operatorlar

Operatorlar ózgeriwshiler hám mánislerde operatsiyalar orınlaw ushın

 isletiledi. PHP operatorları tómendegi gruppalarǵa bólinedi:

  • Arithmetic (Arifmetikalıq)
  • Assignment (Belgilew)
  • Comparison (Salıstırıw)
  • Increment/Decrement (Qosıw/Alıw)
  • Logical (Logikalıq)
  • String (Qatar)
  • Array(Massiv)
