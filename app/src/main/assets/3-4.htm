3.4. Yii2 framework

Yii - bul tez isleytuǵın komponentli PHP framework bolıp, zamanagóy web qosımshalardı tez jaratıw ushın mólsherlengen, joqarı nátiyjeli, komponentlerine tiykarlanǵan, ob'yektke jóneltirilgen baǵdarlama bólimleri kompleksi bolıp tabıladı. Ol ámeldegi koddan maksimal dárejede tákirar paydalanıp, web qosımshanı tezlik penen jaratılıwma múmkinshilik beredi. Baǵdarlama ishki bólimlerin tártipli bolıwına kómeklesedi.

Yii (Yi [ji:] sıyaqlı aytıladı) sózi qıtay tilinde "ápiwayı hám evolyutsiyalaytuǵın " degen mánisti ańglatadı. Sonıń menen birge Yii akronim retinde qaralganda onıń jayılma teksti Yes It Is formasında qaralıwı da múmkin. Yii - bul universal framework jáne onı barlıq túrdegi web qosımshalar ushın qóllaw múmkin. Onıń komponentli strukturası hám keshlewdi júdá zor qollap-quwatlawı ornına framework tiykarınan portallar, forum, CMS, magazinler yamasa RESTful qosımshalar sıyaqlı úlken proyektlar ushın qol keledi.

Yii diń basqa frameworklardan abzallıǵı hám kemshiligi

Ø Basqa kóplegen PHP frameworklar sıyaqlı Yii da kodtı shólkemlestiriw ushın MVC (Model-View-Controller) modelinen paydalanadı.

Ø Yii tek proektlestiriwdiń málim bir qálipine uqsatıp dizayndı qiyınlastırmastan ápiwayı hám elegantlı kod jazıw filosofiyasına tayanadı.

Ø Yii full-stack framework esaplanadı. Sondayaq óz ishine tekserilgen hám ózin jaqsı kórsetken relatsion hám NoSQL maǵlıwmatlar bazası ushın jaratılǵan ActiveRecord, REST API dı qollap quwatlaw, kóp qatlamlı keslew sıyaqlı imkaniyatlardı aladı.

Ø Yii júdá jaqsı keńeyiwi múmkin. Siz tiykarǵı kodtıń qálegen bir bólimin almastırıwıńız yaki sazlawıńız múmkin. Keńeytiriw arxitekturasına boysınıp kodtı basqalar menen jalǵawshı yamasa jámiyetliktiń kodınan paydalanıw múmkin.

Ø Yii diń tiykarǵı maqsedlerinen biri - islew tezligi.

Yii frameworki islewshi ushın web serverde PHP diń keminde 5.1 versiyası bolıwı talap qılınadı. Barlıq imkaniyatlarıdan paydalanıw ushın web server sıpatında apache, server dástúr tili sıpatında php 5.3, maǵlıwmatlar dereklerin basqarıw sisteması sıpatında MySQL 5.5 bolıwı kerek [www.yiiframework.com].

Óz kompyuterińizge ornatıw ushın olardı bólek-bólek yamasa komplektler formasında (denwer, xampp) alıwıńız múmkin. Yii frameworktıń eń jańa versiyasın proekttiń rásmiy saytınan kóshirip alıw múmkin.
