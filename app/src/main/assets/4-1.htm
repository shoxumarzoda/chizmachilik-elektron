4.1. MySQL maǵlıwmatlar bazası haqqında túsinik.

MySQL Ц bul eń ataqlı hám júdá kóp paydalaniletuǵın maǵlıwmatlar bazasın basqarıw sisteması (MBBS) esaplanadı. Bul sistema kútá úlken maǵlıwmatlar menen islew ushın jaratılmaǵan, kerisinshe azmaz kishi kólem degi bazalar menen úlken tezlikte islew ushın jaratılǵan. Onıń tiykarǵı islew sheńberi, saytlar esaplanadı. Házirgi kúnde júdá kóp sayt hám bloklardıń maǵlıwmatları usı MBBS saqlanadı.

MySQL Ц júdá úlken tezlikte isleytuǵın hám qolay esaplanadı. Bul sistemada islew júdáalsat jáne onı úyreniw qıyınshılıq payda etpeydi.

†MySQL sisteması tcx kompaniyası tárepinen, maǵlıwmatlardı tez qayta islew ushın kárxana kóleminde isletiwge jaratılǵan. Keyinirek keń tarqalıp, saytlardıń tiykarǵı bazası retinde jayıldı.

Sorawlar SQL tili arqalı ámelge aırıladı. Bul MBBS relyatsion maǵlıwmatlar baza esaplanadı. Bul degeni baza kesteler, kesteler bolsa ústinlerden dúzilgen.

MySQL MBBS 2 qúyli túrdegi litsenziyaǵa iye. Birinshisi biypul, yaǵnıy MySQLdi kóshirip alıw hám isletiw hesh qanday ǵárejet talap etilmeydi hám GPL (GNU Public Licenseb, GNU) litsenziyasına tiykarlanadı. Ekinshi tıri, GPL shartine kóre, eger siz MySQL kodların qálegen bir dástúrińizde isletiwińiz, bul dástúrińiz de GPL (biypul) bolıwı kerek. Bul bolsa dástúrshige tuwrı kelmeydi. Sonıń ushın, bul dástúrińizdi pullı qılıwıńız ushın MySQL pullı litsenziyasın satıp alıwıńız kerek.

MySQL logotipi delfin esaplanadı. Bul delfindi ismi ЂSakilaї dir. Bul logotipni OpenSource dúzilıwshilerinenn biri Ambrose Twebaze ǵa tiyisli.

MySQL júda kóp operatsion sistemalar menen isley aladı. Bular: AIX, BSDi, FreeBSD, HP-UX, Linux, Mac OS X, NetBSD, OpenBSD, OS/2 Warp, SGI IRIX, Solaris, SunOS, UnixWare, Windows 95, Windows 98, Windows NT, Windows 2000, Windows XP, Windows Server 2000, Windows Vista, Windows 7, Windows 8, Windows 10Е

MySQL shved kárxanası MySQl AB ǵa tiyisli bolǵan, keyinshelik SUN firması MySQL di óziniki etip alǵan. Bir neshshe jıl aldın bolsa, gigant korxonalardan biri Oracle, SUN firmasın satıp aldi hám házirde MySQL Oracle firmasi ónimi sıpatında shıǵıp kelmekte. Bul bir esabta Microsoft SQL Server ge úlken básekeles ekenligin anglatadı.

MySQL bir neshshe serverlerdiń bir bólimi esaplanadı. Mısal ushın, WAMP, AppServ, LAMP, Denwer, Е Kliyentler MySQL serverine belgili bir kitapxanalar arqalı jalǵanadı. MySQL ge tómendegi baǵdarlamalastırıw tilleri baylanıstırılıp islewi múmkin: Delphi, C, C++, Java, Perl, Php, Python, Ruby hám basqalar.
