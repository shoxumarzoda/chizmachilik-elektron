4.2. MySQL maǵlıwmatlar bazası menen isew.

Tómende qanday etip klient programması mysql ge jalǵanıwdı kóremiz. Bul programma járdeminde MySQL-serverge jalǵanıw, SQL - sorawlardı orınlaw hám sol sorawlar nátiyjelerin kórip shıǵıw múmkin. Bul bólekti úyreniw ushın kompyuterińizda utilita mysql ornatılǵan hám MySQL serveri menen baylanısqan bolıwı kerek.

MySQL serverine mysql programması járdeminde baylanısıw ushın paydalanıwshı atınıń hám ádetde parol kirgiziw kerek. Eger server hám klient hár qıylı mashinalarda jaylasqan bolsa, MySQL serveri jumısqa túsirilgen xost atınıń kórsetiw kerek:

shell> mysql -h host -u user -p

 

Sonnan keyin ekranda tómendegi soraw payda boladı: Enter password:, hám sizge óz parolıńızdı kirgiziwińiz kerek boladı. Eger jalǵanıw tuwrı ámelge asqan bolsa, ekranda tómendegi maǵlıwmat hám komanda qatarı belgisi payda boladı mysql>:

Welcome to the MySQL monitor. Commands end with;

or\g.

Your MySQL connection id is 459 to server version:

Type ‘help’ for help.

mysql>

 

Tómendegi mysql> belginiń payda bolıwı  mysql  dástúri iske tayyarlıǵın bildiredi.

Serverden qálegen paytta QUIT komandasın terip shıǵıw múmkin:

mysql> QUIT

Esletpe:

Ádette MySQL lokal mashinaǵa yaǵnıy ornatılǵan bolsa, múrájat parol hám host kiritilmesten, komanda qatarına mysql komandasın kiritiw jolı menen ámelge asırıladı.

Serverge jalǵanǵannan soń komandalar sintaksisin úyrenip shıǵıw ushın bir neshshe ápiwayı sorawlar beriwińiz múmkin.

Ele hesh qanday maǵlıwmatlar bazası tańlanbaǵanı ushın tómendegi keltirilgen sózler ulıwma xarakterge iye.

Tómende serverden versiyasi hám waqıttı soraytuǵın ápiwayı komandanı keltiremiz:

mysql> SELECT VERSION(), CURRENT_DATE;

Bu sorawdı orınlaw mısalında MySQL menen islewdiń tiykarǵı qásiyetlerin koriw múmkin:

·        Serverge jiberilgen komanda, ádette SQL-ańlatpadan ibarat bolıp, izinenen  noqatlı útir keledi. Bul qaǵıydadan sheginiwler bar, máselen QUIT komandasınan soń noqatlı útir qoyılmaydı;

·        MySQL soraw nátiyjesin keste kórinisinde shıǵaradı;

·        Soraw nátijelerinen ibarat kesteni shıǵarqarǵannan soń, mysql  qaytarılǵan  qatarlar sanı hám soraw orınlaw waqıtın kórsetedi. Bul qolay, sebebi server ónimdarlıǵın hám soraw orınlaw effektivligin bahawǵa imkan beredi;

·        Soraw nátiyjeleri hám orınlanıw waqtın  shıǵarǵannan  soń, mysql   jańa   mysql>   qatarın shıǵaradı, bul bolsa Jańa komandalardı orınlawǵa tayarlıǵın kórsetedi.

MySQL komandaları registrge baylanıslı emes, sonıń ushın tómendegi sorawlar bir qıylı:

mysql> select version ()

 current_date;

mysql> SELECT VERSION ()

 CURRENT_DATE;

mysql> Select Version ()

 Current_DATE

 

MySQL bir qatarǵa bir neshshe komandalardı jaylastırıwǵa imkan beredi, lekin olar hár biri noqatlı útir menen tamamlanıwı kerek.
