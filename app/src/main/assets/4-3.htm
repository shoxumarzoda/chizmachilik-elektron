4.3. MySQL maǵlıwmatlar bazasın jaratıw (CREATE DATA- BASE).

Maǵlıwmatlar bazası CREATE DATABASE komandası járedminde jaratıladı.

Komanda sintaksisi:

CREATE DATABASE database_name

Ø database_name — Maǵlıwmatlar bazasına beriletuǵın atı. Keyingi

mısalda db_test maǵlıwmatlar bazasın jaratamız:

CREATE DATABASE db_test

PHP da maǵlıwmatlar bazasın jaratıw:

$sql= CREATE DATABASE db_test;

mysql_query($sql)

MySQL maǵlıwmatlar bazasın óshiriw (DROP DATA-BASE).

Maǵlıwmatlar bazasın óshiriw ushın DROP DATABASE komandasınan paydalanıladı.

Sintaksis:

DROP DATABASE database_name

Bul jerde:

Ø database_name — óshiriw kerek  bolǵan  maǵlıwmatlar bazası atı.

Tómendegi mısalda db_test maǵlıwmatlar bazası óshiriledi:

DROP DATABASE db_test

PHPda maǵlımatlar bazasın óshiriw:

$sql= DROP DATABASE db_test;

mysql_query($sql)

USE.

Kesteler menen islew ushın MySQL ge qaysı baza menen islew haqqında maǵlıwmat beriw kerek.  Bul USE komandası járdeminde ámelge asırıladı:

USE db_name;

Bul jerde db_name — tańlanǵan maǵlıwmatlar bazası atı.

Jaratılǵan db_test bazasın tańlaymız:

mysql> CREATE DATABASE db_test;

Database changed
