5.1. CMS plotformalarında islew.

CMS (content management system — kontentti basqarıv sisteması) — tayar shablonlı sistemalar, maǵlıwmatlardı kiritiw hám dizayn jumıslarına mólsherlengen funksiyalar toplamınan ibarat platforma. Solay etip, CMS tekstli hám multimedialı betlerden ibarat saytlardı qolay hám ańsat jaratıw hámde ózgertiriw (qosıw, redaktorlaw hám óshiriw) imkanın beretuǵın basqarıw sisteması esaplanadı.

CMS saytında jańa betler jaratıw, ónimler katalogın ózgertiw, buyırtpa beriw sıyaqlı processlerdi ámelge asırıw imkaniyatların da beredi. Hár qanday internet saytı HTML tili, CSS kaskadlı stiller kestesi, PHP, JavaScript sıyaqlı web – texnologiyalar hámde MySQL ge uqsas maǵlıwmatlar bazasın basqarıw sistemaları járdeminde jaratıladı. Sayt quramında biz kóretuǵın barlıq materiallar (tekst, súwret, audio, video hám taǵı basqa) usı web-texnologiyalar járdeminde betke shıǵarıladı. Bizge beklili, bunday process úshewlerbu web-dástúrlew tillerin jaqsı ózlestirgen qániygeler tárepinen ǵana ámelge asırıladı. CMS bolsa dástúrlew tillerinen xabarsız paydalanıwshılar da sayt jaratıw jáne onı basqarıw imkaniyatın bere aladı.

CMS platformalar sayt quramın materiallar menen toltırıw, olardı ózgertiriw hám resurslardı basqarıw, jańa betler yaratıw, sayt kórinisi, dizaynın optimallastırıw hám onıń quramın jańalaw imkaniyatların beredi.

 CMS platformalar eki toparǵa bólinedi:

1.   Ashıq kodlı (biypul tarqatılatuǵın), paydalanıwshı tárepinen ishki kodınıń ózgertiriliwine ruxsat beriletuǵın sistemalar: WordPress, Joomla, Drupal, Typo3, OpenCart, Serendipity, Dotclear, ImpressPages, Chamilo, Magento, PrestaShop;

2.   Jabıq kodlı (sawda-satıq maqsetinde tarqatılatuǵın), paydalanıwshı tárepinen ishki kodtıń ózgertiriliwine ruxsat berilmeytuǵın sistemalar: Shopify, 1С-Битрикс, CS-cart, DataLife Engine, BigCommerce , Shop-Script.

CMS platformalar wazıypasına kóre eki túrge bólinedi:

Ø joqarı qánigelestirilgen. Bul sistemalar bólek múmkinshiliklerge iye bolıp, arnawlı belgilengen wazıypalardı sheshiw ushın mólsherlengen platformalar (mısalı, internet dúkanların tashkil et-jumıs: biypul - OpenCart, Magento hám b., kommerciya maqsetinde tarqatiletuǵın, yaǵnıy pullıq - BigCommerce, Shop- Script hám b.);;

Ø universal. Túrli wazıypalardı shehiwge mólsherlengen platformalar (máselen, WordPress, Joomla, Typo3).

WordPress – bul jeke bloglar hám Web - sayt strukturasın jaratıw, sayt maǵlıwmatların redaktorlaw plotforması esaplanadı. Bul platforma MySQL hám de PHP járdeminde isleydi. WordPress 2013-jıldıń Avgust ayında 10 million Web- saytlardıń 22% iyeledi. Házirgi kúnde WordPress Web álemindegi eń úlken blogging sistemalarınan biri esaplanıp, onıń járdeminde 60 millionǵa jaqın Web-saytlar ashılǵan. WordPresstiń birinshi ret keń jámiyetshilikke nátiyjeni ámelde qollanıwı 2003-jıldıń 27-mayında bolǵan. Ol b2/cafelog dástúriniǵ rásmiy dawamshısı esaplanadı.

WordPress sisteması tez-tez jańalanıp turadı hám hár bir yańalanıw málim bir qosımsha funksiyalar jaratılıwına sebep boladı, onan tısqarı aljasıqlar da tuwırlanadı. Bul CMS dıń jaqsı tárepi sonda, admin paneli júdá qolay hám túsinikli bolıp tabıladı.

WordPress de sayt jaratıw ushın qandayda bir shablon saylanadı hám sol shablon tiykarında sayt dúziledi, shablonlar tabıw mashqala emes, sebebi bul CMS shablonları júdá kóp hám ráń báreń bolıp tabıladı. Shablon hám plaginlar bul CMS dıń tiykarı esaplanadı, mudamı WordPress, shablon hám plaginların jańalap barıw kerek.

WordPress CMS tı ornatıwdı baslaymız, dáslep WordPress CMS ti inter-netten kóshirip alıwımız kerek, álbette óziniń Web - saytınan (https://wordpress. org/). Solaay eken, WordPress saytına kiremiz hám oń tárepdegi «Скачать WordPress 4.7.4» túymeshesin basamız (5.1-súwret).

 


5.1-súwret. Wordpress saytı (https://wordpress.org/)

Eki túrli versiyada kóshirip alıwımız múmkin, biri Windows sisteması ushın «.zip» faylı hám ekinshisi Linux sisteması ushın «.tar.gz» faylı, bizge «.zip» kóriniwi kerek.

       Onnan soń, sayt ushın papka ashamız, bul papka «denwer» ornatılǵan papka ishinde bolıwı kerek, tuwrıraǵı tómendegi adreste (Denwer D:\Webserver papkasına ornatılǵan dep alamız.  D:\Webserver\home\

        Bul papka ishinde «infomalaka.uz» deb papka ashamız hám ishinde jáne «www» papkasın ashamız, sonda tómendegi kóriniste boladı:

D:\Webserver\home\ infomalaka.uz\www

Onnan soń bul papka ishine WordPress CMS ti kóshiremiz hám arxivten ashamız, bunıń ushın bizge «Winrar» kerek boladı. Kóshirip alınǵan WordPress faylınıń ústinde tıshqanshanıń oń tárepin basıp «Извлечь в текущую папку» dı tańlaymız. Bunnan soń, wordpress-4.7.4_Ru.zip faylı ashıladı, fayllar «wordpress» atlı papka ishinde ashıladı. Bul bizge tuwrı kelmeydi, sonıń ushın bul papka ishindegi barlıq fayllardı alıp, «www» papkası ishıne taslaymız, sonda támendegi kóriniske keledi.

D:\Webserver\home\infomalaka.uz\www\wp-admin…

Barlıq wordpress faylları ayne «www» ishinde bolıwı kerek. Endi denwer di qosamız, bunıń ushın denwer papkası ishinen «Run.exe» faylın iske túsiremiz. Bul fayl tómendegi adresde boladı:

D:\Webserver\denwer\Run.exe

Denwer di iske tusirgenimizden soń, MySQL MBBS (maǵlıwmatlar bazasın basqarıw sisteması) ta maǵlıwmatlar bazası hám paydalanıwshı payda etiwimiz kerek. Demek, maǵlıwmatlar bazası atı «infomalaka», paydalanıwshı atı hám paroli bir qıylı «123456» kórinisinde bolsın.

Endi WordPress ti ornatamız, bunıń ushın qandayda bir brauzer ashıp (mısal ushın Google Chrome), adresine «infomalaka.uz» deb jazamız, tómendegi kórinisti brauzer ashıp beriwi kerek(5.2-súwret).

5.7-súwret.

Kórip turǵanımızday, sayt jaratıldı, biraq sál ápiwayıraq. WordPress ornatılǵannan keyin, bul CMS tárepinen hár qıylı papka hám fayllar payda etiledi. Biz jaratqan infomalaka.uz papkası ishindegi www papkasında tómendegi fayl hám papkalar boladı:

wp-admin

wp-content

wp-includes

.htaccess

Bulardan tısqrı jáne «wp» dep baslanatuǵın bir qansha fayllar boladı. Joqorıda aytıp ótkenlerimiz benen tanısıp shıǵamız:

Wp – admin - bul papkada sayt admin paneli faylları jaylasadı. Hár bir sayttıń (CMS bolmaǵan saytlarda da ) óz admin paneli boladı, bul arqalı sayt basqarıladı. WordPress CMS de admin panel bolıp, bul panel kórinisi hám basqarılıwın payda etken fayllar sol katalogta saqlanadı. Qısqa etip aytiwımız múmkin, admin panel de bir kishkeni sayt kórinisinde bolıp, tolıq sayt, hátte, maǵlıwmatlar bazası da sol panel arqalı basqarılıwı múmkin. Bul katalogtıń ishin ózgertiwdi usınıs etpeymiz.

Wp-content — sayttıń tiykarǵı katalogı, bul katalogta saytqa tiyisli barlıq fayllar, shablonlar, plaginlar, mediafayllar (súwret, video ) saqlanadı. Bul katalog ishinde tómendegi tiykarǵı kataloglar bar boladı:

languages — CMS WordPress tili (rus tili, ingliz tili, …);

plugins — sayt plaginleri saqlanadı;

themes — sayt shablonları ushın orın.

Wp-includes — bul jerde sistema faylları saqlanadı, yaǵnıy WordPress yadrosı. Bul papkanı ózgertiriw ulıwma múmkin emes, Windows sistemasındaǵı system32 papkasıń o‘zi. Bu yerdagi fayllardı ózgertiriw WordPress ti isten shıǵaradı.

.htaccess — bul fayl arqalı Apache web serverin basqarıw múmkin. Bunnan tiykarınan IP adres-lerdi filtrlew, betlerdi basqarıw (kimge múmkin, kimge múmkin emes), jóneltiriwlwrdi (redirect 301) ámelge asırıw maqsetinde paydalanıw múmkin.

index.php — tiykarınan web bet, saytqa múrájet etilgende, birinshi sol fayl ishine túsedi hám shablonǵa silteme beredi.

Wp-config.php — tiykarǵı sazlawshı fayl, maǵlıwmatlar bazası menen baylanısıwdı sazlaw, CMS dárejesindegi hár túrlı global ózgeriwshilerge baha beriw múmkin. Ulıwma alıp qaraǵanda, bizge tek wp-config.php faylı hám wp-content papkası kerek boladı, qolǵan fayl hám papkalarǵa ulıwma tiymesek de boladı. Ulıwma CMS ti jańalap atırǵanda (update) bul papkalar da jaáalanadı, lekin siziń fayllarıń, jańa wp-content ishindegi fayllar ózgermeydi.

CMS deǵı hár bir sayt málim bir shablonǵa tıykaralanǵan halda tayarlanadı. Basqa CMS shablonları WordPress ke túspeydi (házirshe), sebebi ol shablonlardıń strukturası, WordPress shablonları strukturasına uyqas kelmeydi.

 Joqarıdaǵı pikir menen ne dep aytajaqshımız? Hár bir WordPress shablonları belgili bir ózgermeytuǵın strukturaǵa iye boladı. Shablonda bazı bir fayllar bolıp, bul fayllar tiykarında sayt dúziledi.

WordPress ti úyrengenımizden soń, bizge «twentyfifteen» atlı shablonlı sayt usınıs etiledi. Bunı qayerden biliw múmkin, denwer di iske tusiremiz hám brauzerge «infomalaka.uz» deb jazamız hám payda bolǵan betti HTML kodın kórsetemiz (CTRL+U basıladı), sol kod ishinde «..theme/ twentyfifteen» jazıwlar boladı, sol atama shablon (theme) atı boladı. Bul shablon tómendegi manzil (adres) de jaylasqan boladı: 

../home/test.uz/www/wp-content/themes/ twentyfifteen

 

WordPress ke ornatılatuǵın barlıq shablonlar áne usı papkaǵa jaylasadı.

Bul da barlıq basqa shablonlarda tómendegi tiykarǵı fayllar bar boladı:

archive.php — arxiv maqalalar saqlanatuǵın web betlik fayl;

comments.php — kommentariyaǵa juwap beretuǵın fayl;

footer.php — sayttıń tómengi bólimi ushın juwap beretuǵın fayl;

function.php — saytqa tiyisli barlıq funksiyalar saqlanatuǵın fayl;

header.php — sayttıń joqarı bólimine juwap beretuǵın fayl;

index.php — birinshi júklenetuǵın fayl, tiykarǵı fayl;

page.php — maqalalardı ulıwma web betlikte kórsetetuǵın fayl, bir neshshe maqalalardı bir bettelik kóriniste bul fayl isletiledi;

sidebar.php — sayttıń bazı bir tárepin (oń, shep) payda etetuǵın fayl;

single.php — bir maqalanı kórsetiwde isletiletuǵın fayl, yaǵnıy maqalanı tolıq kórgende;

style.css — sayt stilleri saqlanatuǵın fayl;

search.php — izlew nátiyjeleri betlik ushın fayl;

404.php — betlik yaki maǵlıwmat tabılmaǵanda, usı fayl isleydi.

Joqarıdaǵı fayllar barlıq shablonlarda ámelde bar bolıp, tek ishindegi kodları ózgeriwi múmkin, tiykarıbir qıylı boladı. WordPress CMS tiń dúzilisin tómendegi súwrett kóretuǵın bolsaq (5.8-súwret):

5.8-súwret. WordPress CMS tiń dúzilisi

 

WordPress CMS diń admin paneli, hár bir sayttıń admin paneli bolıwı kerek, bul admin panel arqalı sayt kodlarısız basqarıladı hám kliyent ushın qolaylı esaplanadı. Kliyent kodlardı túsinbeydi, sol sebepli olar ushın grafikalıq admin panel bolıwı kerek. WordPress de admin panel júdá qolay hám túsinikli etip dúzilgen, programmalastırıwdı bilmey turıp da bul CMS de dúzilgen sayttı basqarıwımız (maqalalar qosıwıńız, kommentariyalerdi baqlawımız hám basqalar ) múmkin.

WordPress admin paneline kirisiw ushın, dáslep denwer di iske túsiremiz hám brauzerge sayt_nomi/wp-admin adresin kiritemiz mısal ushın: infomalaka.uz/wp-admin. Onnan soń payda bolǵan web betke sayt asıp atırǵanda kiritket login hám paroldi jazamız, «Войти» túymeshesin basamız(5.9-súwret).

Настройки -> Постоянные ссылки

Общие настройки – bul eń tiykarǵı sazlaw esaplanadı, yaǵnıy URL kórinisin sazlaw. Eger bunı sayt internetke qoyılıwdan keyin sazlasańız, biraz qıyınshıliqlarǵa dush keliwińiz múmkin. Bul sazlawdı ЧПУ dep de ataydı, yaǵnıy «Человеко-понятный урл». Mısal ushın biraz sayttaǵı maqalaǵa kirsek onıń adresi http://infomalaka.uz/?p=12s=14 kórinisinde bolsa, adreske qarap bul maqala ne haqqında ekenligin bile almaymız, «?p=12s=14» tan neni biliw múmkin, bul paydalanıwshılar ushın qolaysız, hám izlew sistemaları (google, yandex) bunday URL ler    bilan yaxshi chiqisha olmaydi. Bul degeni, izlew sistemalarınıń, siziń sayttı indekslew processin (google bazasına biziń sayt túsiw itimalı) ástenlesedi degeni esaplanadı.

Eger biz kirgen maqala, tómendegi kóriniste bolsa:

http://infomalaka.uz/CMS/wordpress ti-ornatıw-haqqında biz URL manzilinen gáp ne haqqında barıwın bilip alıwımız múmkin, bul qolay, tusikli hám ájayıp bolıp tabıladı. Bul aynada áyne usı URL kórinisi sazlanadı, WordPress ózi bizge usınıs qıǵan URL adresi, bizdegi birinshi kórinisi bolıp tabıladı, yaǵnıy «/?p=123», bizge bul kerek emes, onnan tısqarı tómendegi basqa kórinisler de bar, biz bul jerge ózimiz URL qanday shıǵarıwdı kórsetemiz. Bunıń ushın «произвольно» bántine ótip, tómendegilerdi jazamız: /%category%/%postname%/

Bul degeni, dáslep bólim atı keyin maqala atı shıǵıw degeni bolıp tabıladı, bul shıǵıwǵa mısallar keltiremiz:

http://infomalaka.uz/CMS/wordpressni -haqqında

http:// infomalaka.uz/Internet/github –ne zat

http://infomalaka.uz/dátúrlew/web/php/massivler haqqında

 

Eger tema atları kiril háriplerden ibarat bolsa, bul jerde biraz qıyınshılıq bolıwı múmkin, bunıń jolı hám bar, bunıń ushın plagin qoyıw kerek , boladı.
