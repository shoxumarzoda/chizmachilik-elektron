KIRISIW

Zamanagóy bilimlendiriw sistemasınıń tiykarın sapalı hám joqarı texnologiyalı ortalıq quraydı. Onıń jaratılıwı hám rawajlanıwı texnikalıq tárepten quramalı, usınıń menen bir qatarda bunday ortalıq bilimlendiriw sistemasın jetilistiriwge, tálim procesine informaciya komunikatsiya texnologiyaların engiziwge xızmet etedi. Bilimlendiriw processinde Web texnologiyalardan paydalanıw shınıǵıwlardı interaktiv rejimde alıp barıwǵa múmkinshilik beredi.

Ásirese, «Ózbekstan Respublikası Joqarı bilimlendiriw sistemasın 2030 jılǵa shekem rawajlandırıw konsepsiyasi»de bolajaq oqıtıwshılardı social-pedagogikalıq iskerlikke tayarlaw máselesi menen qatar zárúrli jónelisler belgilep berildi.

Sol kózqarastan, búgingi kúnde pedagog kadrlardan tekǵana óz tarawı boyınsha, bálki zamanagóy informaciya texnologiyalarınan da málim bilimlerdi tereń iyelewlerin hám olardı jaslarǵa, ásirese oqıwshı hám de talabalarǵa úyretiwlerin talap etedi. Bunnan tısqarı ulıwma bilim beriw mektep, KÓK hám JO Orınlarında iskerlik kórsetip atırǵan hár bir professor-oqıtıwshılarǵa zamanagóy web texnologiyalardan paydalanıp web-dástúrlewdi kompyuterde orınlawǵa úyretiw wazıypasın júkleydi.

Bul qóllanba úsh bólekten ibarat bolıp: birinshi bólekte HTML5, CSS3, Javascript hám JQuery texnologiyası hám olardıń múmkinshilikleri haqqında sóz baradı. HTML5 ning tiykarǵı elementleri, formalar, kesteler, kesteler hám semantik web teglerin úyreniw. CSS3 járdeminde teglerdi formatlawdı úyreniw. JavaScripttiń  tiykarǵı  túsinikleri:  ózgeshelikler, metodlar, hádiyseler. JQueryde selektorlar hám kitapxanasın jalǵaw haqqında tolıq toqtap ótilgen.

Ekinshi bólim PHP texnologiyası hám MySQL maǵlıwmatlar bazasın basqarıw sistemalarınıń (MBBT) múmkinshilikleri hám paydalanıw ushın funksiyalar kompleksi, PHP texnologiyasınıń web sistemalarda paydalanıwǵa arnalǵan. Bul bólimde PHP hám MySQL járdeminde dinamikalıq web-sayttı joybarlaw hám programma kodları mısallar járdeminde túsintirip ótilgen.

Úshinshi bólim zamanagóy CMS programma (Joomla, WordPress, Drupal) ları hám olardı web-serverge jaylaw. LMS plotforma (Moodle, Google Classroom, Mooc) lardıń túrleri hám wazıypaları. LMS tiykarında aralıqtan tálim alıw. Web-freelance hám Web-freelance saytları.
